watchdog 行为分析

100 轮健康对话期间，watchdog 未触发（正确行为）：

  • 每轮：bff.accepted → dispatch.started → dispatch.result(success) → reply.first → reply.progress → terminal
  • 所有 run 在 2-15 秒内完成，远低于 10s/30s/120s 阈值
  • watchdog 每 2s 扫描活跃 run，无超时 → 不触发 → 不误报

A11 freeze 注入测试（独立运行）：

  • 第 26 轮发消息后调 POST /api/debug/inject-stall 冻结 recorder
  • BFF 记录 bff.accepted → dispatch.started → dispatch.result 后 freeze 生效
  • reply.first/terminal 无法记录 → watchdog 120s 后触发 watchdog.terminalMissing
  • incidentCollector 落盘现场包
