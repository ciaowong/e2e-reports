收不到验证码

调用链摘要

前端 → 身份服务 → 短信平台 + 验证码存储

登录和短信验证码不经过业务中台，优先看身份服务侧的时间窗和状态信号。

用户现象 优先判断 归属 公开信号
点击获取验证码后无短信 身份服务是否触达短信平台 我方 / 外部 时间窗内是否出现短信平台错误；平台拒绝偏外部，网络不可达偏我方配置或网络。
提示请求频繁或图形码错误 是否被前置校验拦截 正常保护 同一时间窗内身份服务返回频控或校验失败状态。
验证码已发但登录提示过期 验证码存储是否可用、是否重启丢失 我方 验证码存储和过期时间需要内部命令确认，公开版不展示具体查询方式。

进不去工作台

调用链摘要

前端 → 业务中台 → 工作台控制服务 → 存储资源 → 沙箱资源 → 工作台服务

这条链路要区分“没分配资源”“沙箱没起来”“服务起来但不可用”三类问题。

用户现象 优先判断 归属 公开信号
进入页面长时间等待 控制面分配流程是否超时 我方 业务中台或控制服务出现分配超时类错误；内部版用用户标识追踪。
沙箱没有创建成功 外部沙箱资源或配置是否异常 基础设施 控制面能区分资源不足、鉴权失败、配置缺失等方向。
沙箱存在但文件空间异常 个人存储空间是否分配成功 基础设施 / 配置 诊断结果会标出存储挂载相关根因；公开版不展示内部字段。
资源都有但仍不可用 工作台服务进程或健康检查是否通过 我方 内部诊断按连通性、配置、进程、健康状态分步返回。

发消息没回复

调用链摘要

前端 → 业务中台 → 工作台通道 → 工作台引擎 → 模型服务 → 前端推送

先分清“发不出去”“工作台没处理”“模型失败”“回复回不来”。

用户现象 优先判断 归属 公开信号
发送后一直转圈 业务中台是否找到可用工作台 我方 找不到运行中工作台或通道凭据时，先回到工作台诊断。
能发出但无最终回复 工作台服务或模型服务是否异常 我方 / 外部 内部诊断可分模型连通、通道连通和工作台健康。
回复已生成但页面没显示 前端连接是否在线 用户网络 / 客户端 服务端能看到推送目标离线或连接鉴权失败类信号。
完整值班版保留在内部任务文档中，包含精确命令、日志变量、访问边界、风险分级和待补环境信息。公开版只用于远程快速查看排障思路。
